## Conversion Funnel

```
Awareness → Interest → Signup → Analysis → Consultation

Track:
- Signup rate: Target 15-20%
- Analysis completion: Target 70%
- Consultation booking: Target 30% of completed
- Repeat consultation: Target 40%
```

---

## Key Metrics

- Monthly Active Users (MAU): Target 5K by Month 3
- Consultation rate: 30% of signups
- Revenue/user: NPR 500-1000/month
- CAC (Customer Acquisition Cost): < NPR 200
- LTV (Lifetime Value): > NPR 5000
